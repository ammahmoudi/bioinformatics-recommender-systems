# Image Replacement Prompts

Use one consistent image style for all generated assets:

**Global style:** Fluent 3D educational icon, soft rounded geometry, clean high-detail object, subtle glass material, cyan/mint/amber/rose accents, solid matte background color `#101820`, centered subject, no text, no labels, no logos, no watermark, strong object-background separation for easy background removal.

**Recommended export:** PNG, `1024 x 1024`, 1:1 aspect ratio, transparent background if your generator supports it. If not, use the solid `#101820` background from the prompt.

## CSS Replacement Pattern

Generated images can replace placeholder backgrounds without changing the slide layout:

```css
.mediaMark.poster.inceptionImage {
  background-image:
    linear-gradient(rgba(7,16,22,.08), rgba(7,16,22,.22)),
    url("/assets/movie-inception-node.png");
  background-size: contain;
  background-position: center;
  background-repeat: no-repeat;
}
```

For per-node images, add a specific class to the relevant `FlowNode` or `MediaMark` in `src/diagrams/RichFlow.tsx`, then target that class in `src/styles/diagrams.css`.
