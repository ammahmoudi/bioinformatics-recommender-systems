4. Graph-based methods

Recommend by walking through a network.

Graph methods use paths: user to movie to actor to another movie, or disease to gene to protein to candidate target.

Integrated placeholders

The actor and movie image placeholders are represented inside the graph nodes. Use `image-prompts.md` if you want to replace those nodes with generated images.

pt+1 = (1-c) W pt + c p0

Why restart?

Without restart, the walker may drift far away and recommend generally popular nodes. Restart keeps the score personalized around the starting user, disease, or drug.

Tools

PageRank, Personalized PageRank, Neo4j Graph Data Science, NetworkX, igraph, STRING, BioGRID, Cytoscape, Hetionet, OpenTargets.

6. Graph neural networks

Nodes update by receiving messages.

Atoms pass messages over bonds. A node keeps its own features through self-loops while also aggregating neighbor information.

Self-looped adjacency

Ã = A + I. Normal graph edges plus each node's edge to itself, so every node keeps its own current features while receiving neighbor messages.
